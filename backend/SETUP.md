# ClipChopper Backend — Environment Setup

Copy this file to `.env` and fill in your values.

```
APP_NAME=ClipChopper
APP_ENV=local
APP_KEY=                          # generated by: php artisan key:generate
APP_DEBUG=true
APP_URL=http://localhost:8000
FRONTEND_URL=http://localhost:4321

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=clipchopper
DB_USERNAME=root
DB_PASSWORD=YOUR_MYSQL_PASSWORD   # ← fill this in

# Stripe (get keys from dashboard.stripe.com)
STRIPE_KEY=pk_test_...
STRIPE_SECRET=sk_test_...
STRIPE_WEBHOOK_SECRET=whsec_...
STRIPE_PRICE_MONTHLY=price_...
STRIPE_PRICE_ONE_TIME=price_...

JWT_SECRET=                       # generated by: php artisan jwt:secret
```

## Quick Start

```bash
cd backend

# 1. Copy env
cp .env.example .env              # (on Windows: copy .env.example .env)

# 2. Fill in DB_PASSWORD in .env

# 3. Create the MySQL database
#    Open MySQL Workbench or run:
#    mysql -u root -p -e "CREATE DATABASE clipchopper;"

# 4. Run migrations & seed admin user
php artisan migrate --seed

# 5. Start API server
php artisan serve
# → API available at http://localhost:8000/api/v1/
```
